# 飞书应用部署指南

## 📦 应用信息

- **应用名称**：电商管理系统
- **应用类型**：企业自建应用
- **部署方式**：网页应用（iframe 嵌入）

---

## 🚀 部署步骤

### 1. 上传构建文件到服务器

**方式一：使用 Vercel（推荐，免费）**

```bash
# 安装 Vercel CLI
npm install -g vercel

# 进入 dist 目录
cd dist

# 部署到 Vercel
vercel --prod
```

部署后会获得一个 URL，例如：`https://your-app.vercel.app`

**方式二：使用自己的服务器**

将 `dist` 目录的所有文件上传到服务器的 web 目录（如 Nginx 的 `/var/www/html`）

---

### 2. 在飞书开发者后台创建应用

1. 访问 [飞书开发者后台](https://open.feishu.cn/app)
2. 登录企业管理员账号
3. 点击 **"创建企业自建应用"**
4. 填写应用信息：
   - **应用名称**：电商管理系统
   - **应用图标**：上传一个图标（建议 512x512 PNG）
   - **应用描述**：电商商品、采购、供应商管理系统

---

### 3. 配置应用访问地址

1. 在应用管理页面，点击 **"应用配置"**
2. 找到 **"网页应用"** 或 **" iframe 嵌入"** 选项
3. 填写：
   - **PC 端访问地址**：`https://your-app.vercel.app`（或你的服务器地址）
   - **移动端访问地址**：同上
4. 点击 **"保存"**

---

### 4. 配置应用可见范围

1. 点击 **"权限管理"**
2. 配置应用可见范围：
   - 选择需要访问应用的部门或人员
   - 或者设置为 **"全公司可见"**
3. 点击 **"保存"**

---

### 5. 发布应用

1. 点击 **"版本管理"**
2. 点击 **"创建版本"**
3. 填写版本信息：
   - **版本号**：1.0.0
   - **版本描述**：初始版本
4. 点击 **"提交审核"**（企业自建应用通常不需要审核）
5. 审核通过后，点击 **"发布"**

---

### 6. 添加到工作台

1. 发布后，应用会出现在飞书工作台中
2. 员工可以在飞书工作台看到并打开应用

---

## 🔧 飞书 SDK 集成（可选）

如果需要获取飞书用户信息、发送消息等，需要集成飞书 SDK：

### 1. 安装飞书 JS SDK

```bash
npm install @lark-base-open/js-sdk
```

### 2. 在 main.js 中初始化

```javascript
import { api } from '@lark-base-open/js-sdk'

// 获取当前用户信息
api.user.getUserId().then(res => {
  console.log('用户 ID:', res.userId)
})
```

### 3. 配置飞书应用权限

在飞书开发者后台，添加以下权限：
- `user:userId` - 获取用户 ID
- `contact:contact.readonly` - 读取通讯录

## 📊 数据持久化说明

**当前存储方式**：localStorage（浏览器本地存储）

**优点**：
- 无需服务器
- 快速部署
- 数据在本地，安全

**缺点**：
- 数据只在当前浏览器
- 切换设备数据不同步
- 清除浏览器数据会丢失

**升级建议**（后续）：
- 对接飞书云基地（Bitable）存储数据
- 实现多人协作和数据同步
- 数据永久保存

## 🆘 常见问题

**Q: 应用打不开？**
A: 检查访问地址是否正确，确保使用 HTTPS

**Q: 看不到应用？**
A: 检查应用可见范围配置，确保你的账号在可见范围内

**Q: 数据丢失？**
A: localStorage 数据在当前浏览器，清除缓存会丢失。建议后续对接飞书云基地
